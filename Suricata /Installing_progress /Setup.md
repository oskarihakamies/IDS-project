# Suricata Installation to VM

### First Step 

### Setup 

- VM (Virtual Box, UTM)
- Debian 13
- Atleast 4 CPU cores and 20 GB RAM

### Second Step

When you have an VM this is how you install and configure suricata.

Make basic updates and upgrades:

```
sudo apt update
```
```
sudo apt upgrade -y
```


And then install it
```
sudo apt install suricata -y
```
